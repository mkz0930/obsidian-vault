# MyDevBrain - Personal Knowledge Management System

## 目录结构

```
MyDevBrain/
├── Daily/           # 每日笔记（Calendar 插件自动生成）
├── Projects/        # 所有项目
│   └── Project-A/   # 示例项目
│       ├── README.md
│       ├── Progress.md
│       ├── Decisions.md
│       ├── Notes/
│       └── Code-Snippets/
├── Learning/       # 学习内容
│   ├── Courses/
│   ├── Books/
│   ├── Tech-Topics/
│   └── Insights/
├── MOC/            # 地图（Maps of Content）
├── Templates/       # 模板
└── Archive/        # 已完成的项目
```

## 项目规范

### Projects/
- `README.md` - 项目总览 + 目标
- `Progress.md` - 进度追踪（最重要）
- `Decisions.md` - 架构决策记录 (ADR)
- `Notes/` - 项目相关零散笔记

### Learning/
- 按课程/书籍/技术主题/个人洞见分类

## 使用技巧

- 使用 `[[双链]]` 连接笔记
- 用 `#标签` 标记主题
- MOC 用于聚合相关主题笔记
